本插件使用方法：
---------------
以整合完整安装的《武装突袭2》为例:
1、将本压缩包中的“@vme_clp”文件夹解压至游戏安装目录下；
2、在游戏安装目录下找到游戏执行程序“arma2oa.exe”；
3、鼠标右键点击游戏执行程序并拖动至Windows桌面，在随后弹出的快捷菜单中选择“在当前位置创建快捷方式”；
4、鼠标右键点击刚刚生成好的快捷方式，并在弹出的快捷菜单中选择“属性”；
5、在“快捷方式”选项卡的“目标”文本框中游戏执行路径的最后（双引号后）键入一个空格，然后输入启动参数“-mod=BAF;PMC;@vme_clp”（BAF和PMC是DLC，请视个人游戏安装情况决定是否填写）；
6、点击快捷方式进入游戏即可显示中文。

补充说明：
---------
1、本插件已删除了其他语言文字，所以请在游戏设置中选择游戏语言为“English（英文）”；
2、使用本插件后可以支持游戏中的中文输入和显示（对方需同样使用本插件），但在全屏状态下无法看见输入法状态，所以如需输入中文请使用启动参数“-window”在窗口模式下运行游戏；
3、本插件中文字符涵盖完整的GBK中文字符部分，可以完美同时显示简体中文和繁体中文；
4、本插件的更新信息请查看：http://www.chinavme.com/thread-14518-1-1.html
